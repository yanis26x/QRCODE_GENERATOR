# QR Code Generator

Ce programme permet de générer un code QR à partir d'une URL personnalisée. Il est simple à utiliser et ne nécessite que quelques modifications avant exécution.

## Utilisation

Suivez ces étapes pour générer votre propre code QR :

1. Accédez au fichier `RUN_THIS.java` situé dans :
   ```
   main/java/yanis26x
   ```
2. Modifiez l'URL à la ligne **37** en remplaçant l'URL par celle de votre choix.
3. Exécutez le fichier `RUN_THIS.java`.
4. Une fois le fichier exécuté, rendez-vous dans le dossier `data`.
5. Récupérez l'image du code QR qui a été créée sous le nom :
   ```
   QR_CODE.PNG
   ```

Et voilà ! Votre code QR est prêt à être utilisé.

# QR Code Generator (English Version)

This program generates a QR code from a custom URL. It is simple to use and requires only a few modifications before execution.

## Usage

Follow these steps to generate your own QR code:

1. Navigate to the `RUN_THIS.java` file located in:
   ```
   main/java/yanis26x
   ```
2. Modify the URL at **line 37** by replacing it with your desired URL.
3. Run the `RUN_THIS.java` file.
4. Once the file has been executed, go to the `data` folder.
5. Retrieve the QR code image that was created, named:
   ```
   QR_CODE.PNG
   ```

And that's it! Your QR code is ready to use.
